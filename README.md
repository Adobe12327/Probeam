# Probeam
Probeam is new Jbeam editor for blender.

https://www.beamng.com/threads/probeam-new-jbeam-blender-plugin.86283/

## Advantages
* Jbeam Import/Export
* Export Full Jbeam (Node, Beam, Triangle)
* Can edit all modifiers with UI
* Familiar blender controlls
* etc.
